## 项目规范说明
+ node version： 16.17.1

该项目使用pnpm包管理工具进行管理，项目支持tsx以及template语法，基础结构由tsx搭建完成，建议开发语法统一
项目已支持eslint与prettierc，以及commit提交规范限制，在项目提交前会统一自动化格式代码，并校验commit语法规范，
推荐使用cz代替 git commit，因为此插件提供友好的交互提示并能生产规范的commit的信息
====项目git提交流程=====
```sh
git add .
npx cz
```

## 项目 Setup

```sh
pnpm install
```

### 启动服务

```sh
pnpm dev
```

### 打包

```sh
pnpm build
```
